# Nacho Juan's Dotfiles

Dotfiles managed by [GNU Stow](https://www.gnu.org/software/stow/).

## Quick Start

1. Install Brew
2. Install Stow
3. Clone the repo to your home directory
4. Run the next command on the `dotfiles` repo

```bash
stow . -t ~
```
